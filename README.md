
`todo`
=========

Make command-line todo lists available in multiple locations.

- On the internets
- On another computer
- In another directory

Setup
-----------

make install
  - list
  - of
  - .rcfile-param prompts

Use
--------

    todo init <filename>
    todo <filename>
